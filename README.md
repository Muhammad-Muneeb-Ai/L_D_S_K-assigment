# 💰 Income Predictor Pro

A Machine Learning web app that predicts whether a person's annual income is **>50K or ≤50K** based on US Census demographic data.

## 🚀 Features
- 3-tab input form (Personal, Work & Education, Financial)
- SVC model with **86.29% accuracy**
- Clean, responsive UI built with Streamlit

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **ML Model:** Scikit-learn (SVC)
- **Data:** UCI Adult Census Income Dataset

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Model
- Algorithm: Support Vector Classifier (SVC)
- Accuracy: 86.29%
- Dataset: adult.csv (UCI Census Income)
